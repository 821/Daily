## div.py
把高級 div 全部剝掉，衹剩下最底層 div 。實際上是個整理辭書的腳本。需要 Beautiful Soup 。
輸入文件一行一個 HTML ，不需要處理的內容放在前方，需要的放後面，二者之間以 \`\`\` 隔開。
需要 Beautiful Soup 。

## zd.bat
1. 爲所有子文件夾建立一個不壓縮的 zip 。
2. 特色在於，是進每個文件夾內，全選後打包而成的。
3. 需要以默認參數安裝的 WinRAR 。

## zs.bat
1. 爲所有子文件夾建立一個壓縮率最高的 zip 。
2. 特色在於，是進每個文件夾內，全選後打包而成的。
3. 需要以默認參數安裝的 WinRAR 。
